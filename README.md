# Hero io

Hero io is a web app that allows users to manage and explore applications with ease. Users can install, uninstall, search, and sort apps directly from the platform.

## Features

1. **Install Apps**
   Users can install apps, and a toast notification confirms when the installation is successful.

2. **View Installed Apps**
   Users can see all the apps they have installed from the installation page.

3. **Live Search**
   Users can search for apps in real-time from the apps page using the live search feature.

4. **Sort Apps by Downloads**
   On the installation page, users can sort apps based on the number of downloads.

5. **Uninstall Apps**
   Users can uninstall apps from the installation page, with a toast notification confirming successful uninstallation.
